# Examples

These examples are documentation-only recipes that point at committed eval
fixtures. They are safe to read and run from a fresh clone.

## Available Examples

- [Validate Trace](validate-trace/README.md): validate, replay, query, and export
  a committed trace fixture.
- [Direct Runtime Team](direct-runtime-team/README.md): understand the
  `spawn_agent` lifecycle contract without needing a live runtime.
- [Auto Harness Foreground](auto-harness-foreground/README.md): inspect a
  foreground baseline-plus-iteration fixture.

## Run All Repo Evals

```bash
python3 scripts/run_harness_evals.py
```
