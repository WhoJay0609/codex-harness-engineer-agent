# Code of Conduct

Be direct, technical, and respectful.

This project is about making agent work easier to inspect and improve. Keep
discussion focused on reproducible behavior, evidence, and concrete changes.

Unacceptable behavior includes harassment, personal attacks, leaking private
artifacts or credentials, and intentionally disruptive issue or pull-request
activity.

Maintainers may close or hide comments that do not help the project move toward
clearer, safer, more reproducible harnesses.
